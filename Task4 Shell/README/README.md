Реализация простого Unix‑подобного интерпретатора shell на Си.

## Функционал

- Лексический анализ
- Построение синтаксического дерева команд
- Выполнение команд:
  - Внешние: `ls`, `cat`, `pwd`, `sleep` и др.
  - Встроенные: `cd`, `echo`
- Поддержка:
  - Перенаправлений: `>`, `>>`, `<`
  - Пайплайна: `|`
  - Последовательного выполнения: `;`
  - Фонового режима: `&`
  - Сабшелла: `()`
  - Логических операторов: `&&`, `||`
